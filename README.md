# Oregon Trail - A C# Refactoring Exercise

Contained in this repository is a conversion of the original Oregon Trail basic program (or, at least, a very early version).

The source code is as close as possible to the original, within the limitations of the C# syntax.

This file is intended as the starting point for a refactoring exercise to practice refactoring tools and techniques. The aim is to convert the code into a well-structured, idiomatic, C# application. It is recommended to introduce tests as you go, to allow safer refactoring.
